# Lambourne House Block Theme Starter

This starter follows the WordPress block theme structure:

- `style.css`
- `theme.json`
- `templates/index.html`
- `templates/front-page.html`
- `parts/`
- `patterns/`
- `assets/`

It is designed to match the official WordPress theme structure documentation for block themes.

## What To Upload

If someone with theme-install access is uploading from WordPress admin, use the ZIP created from this folder.

## Homepage Setup

After the theme is activated:

1. Create a page called `Home`
2. Go to `Settings -> Reading`
3. Set the homepage to that `Home` page

Because this is a block theme, the homepage layout is provided by `templates/front-page.html`.
